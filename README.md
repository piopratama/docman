# Docman PBL Project

[![DOI](https://zenodo.org/badge/689923106.svg)](https://doi.org/10.5281/zenodo.18732451)

## Description

Docman is a Project-Based Learning (PBL) project designed to manage documents through a web-based platform. It is built using a custom PHP framework developed specifically for this project, providing a structured and efficient document management system.

## Features

- User authentication  
- Document upload and download  
- Dashboard for document management  
- Role-based access control  

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/docman.git
   ```

2. Navigate into the project directory:
   ```bash
   cd docman
   ```

3. Install dependencies:
   ```bash
   composer install
   ```

4. Create a `.env` file and configure your environment variables.  
5. Run database migrations (if applicable).  
6. Start the PHP development server.

## Usage

After installation:

1. Configure your database connection in `.env`.
2. Access the application via your local server.
3. Log in using your registered credentials.
4. Manage documents through the dashboard interface.
